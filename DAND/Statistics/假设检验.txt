均值的95%置信区间（confidence interval）

样本量增大，总体参数在干预后的95%置信区间减小，越能精确估计总体参数

margin of error 误差界限


单尾检验（可在左端，可在右端）

alpha 水平
alpha = 0.05 (5%)  -> Z临界值 1.65
alpha = 0.01 (1%)  -> Z临界值 2.33
alpha = 0.001(0.1%)-> Z临界值 3.08


双尾检验（Two tails test）
alpha = 0.05(5%)是两端的，一端则是 0.025(2.5%) -> Z临界值 正负1.96
alpha = 0.01(1%)是两端的，一端则是 0.005(0.5%) -> Z临界值 正负2.57
alpha = 0.001(0.1%)是两端的，一端则是 0.0005(0.05%) -> Z临界值 正负3.32



零假设 H0(Null Hypothesis)：总体均值与在某种干预后出现的新总体均值之间没有显著差异

样本均值将位于临界区之外

对立假设 HA(Alternative Hypothesis)：总体均值与干预后的新总体均值会有显著差异

样本均值位于临界区之内


单尾检验（方向性假设检验）
双尾检验（非方向性假设检验）

一般会选择双尾检验，因为其更为保守。当为真时，不太可能会拒绝零假设

单尾检验需要预测处理措施的方向，有可能与预期背道而驰但没有处理措施；当然，如果只需要考虑新措施的正面效果，可使用单尾检验


假设检验步骤：
1、选择单尾检验or双尾检验
2、确定alpha水平（一般是0.05），得出Z临界值
3、算出样本均值的Z值
4、检查是否位于临界区内
5、是拒绝还是无法拒绝零假设

拒绝零假设条件：
（1）样本均值会落在临界区内
（2）Z值要大于Z临界值
（3）获得样本均值的概率要小于alpha水平

相依样本t检验

1、重复衡量设计（零假设是指这两个总体均值将相同）
为同一个人提供两种条件，看看他们对这两种条件的反应
这两种条件可以是对照组和处理组，或者两种处理类型

2、纵向设计，在一个时间点衡量某个变量，然后在晚些时候的某个时间点再衡量同一变量
在时间1和时间2时，该变量是否存在显著差异

在样本量为n的情况下：
时间1时，均值为miu1，标准偏差为s1
时间2时，均值为miu2, 标准偏差为s2

单尾t检验：
零假设H0: miu2 - miu1 <= 0
对立假设Ha: miu2 - miu1 > 0

则：
差值均值miuD = miu2 - miu1
差值标准偏差 sD = SQRT(s1^2 + s2^2)

t = (miuD - 0) / (sD / SQRT(n))


3、预期检验和后期检验，先衡量某个变量，然后进行某种处理，在处理后再对同一样本衡量同一变量
查看处理措施是否导致显著效应（零假设是处理前后该变量没有出现显著变化）


点估计 -> 基于样本

差异点估计 ―> 两个样本的均值差
差异的标准偏差 -> 样本之间的差值的标准偏差

调查研究的一个重要方面就是 效应量（Effect Size）

在实验性研究中或存在处理变量的研究中，效应量是指处理效应的大小
在非实验性研究中，效应量是指变量之间的关系强度 

在z检验或t检验中，最简单的效应量衡量指标是 均值差异（即两个均值之间的差异）

当我们不知道变量的含义是什么的时候，均值差异可能并不是信息最充分的度量方法

效应量分类：
1、差异度量 Difference measures
（1）均值差异 mean difference（最简单的）
（2）标准化差异度量 Standardized differences
     Cohen's d  （实际上度量的是均值差异，并用标准偏差作为单位）

2、相关度量 Correlation measures
（1）r^2  ―― 表示的是某个变量的变化比例或百分比与另一个变量的关系（有时候会用“解释”而非“相关”，“解释”是指通过知道另一个变量的值能够解释某个变量的特定变化百分比或比例）


统计显著性 statistical significance：
1、显著性表示拒绝了零假设 reject the null hypothesis
2、结果不太可能是偶然发生或抽样错误(sampling error)导致的  （即，我们在解释结果时，排除了随机因素或抽样错误）


结果的意义 Meaningfulness of Results:
1、度量了什么？变量是什么？
变量存在实际practical 或社会意义social 或理论意义theoretical

2、效应量 effect size

3、能对结果排除随机因素吗？（在解释时能排除抽样错误吗？）
这并不能保证结果很重要，但可以帮助评估重要性的其他内容

4、能排除结果的对立解释吗？ (潜在变量 lurking variables)



Cohen's d： 以样本的标准偏差作为单位，这些均值之间相差多少

d = (X - miu) / S


r^2: 确定系数 coefficient of determination

由于r^2是两个变量间的比例关系，取值范围[0,1]。 0表示完全不相关，1表示完全相关（现实中很少见）

在进行t检验时，r^2的公式：
r^2 = t^2 / (t^2 + df)   

其中，df -> degrees of Freedom，t值并不是临界值而是从t检验中获得的值



报告：
1、描述统计量 Description Statistics （例如均值M，标准偏差SD）
描述统计量是研究的核心，告诉我们研究中发生的情况

展现形式
（1）文本形式 in text，即在结果部分直接写出来
（2）用图表 in graph，例如条形图
（3）表格形式 in table，

2、推论统计量 Inferential Statistics （假设检验Hypothesis test、置信区间confidence interval，或两者均有）

（1）假设检验 （alpha水平）：
（1.1） 检验类型 kind of test，例如单样本t检验
（1.2） 检验统计的实际值，例如 t值
（1.3） 检验的自由度，实际值
（1.4） P值，尽可能提供精确的P值
（1.5） 检验方向，单尾检验还是双尾检验

使用格式： APA style （American Psychological Association）
例子：t(df) = X.XX, p=.xx, direction

（2）置信区间
（2.1）置信区间 confidence level， 例如 95%
（2.2）下限 Lower Limit
（2.3）上限 Upper Limit
（2.4）是什么的置信区间 CI on what? （置信区间的用途））
 例如，是单个均值的置信区间，还是两个均值之间的差异的置信区间

使用格式： APA style
例子：Confidence interval on the mean difference; 95% CI = (4,6)

（3）效应量度量结果 Effect Size Measures （Cohen's d, r^2, or both）

APA Style:   d = X.XX or r^2 = .XX

相依样本 Dependent samples(repeated measures)
针对受试者内设计

好处：
1、控制个体差异性
2、使用较少的受试者
3、成本更低
4、花费时间更少
5、开支更少

不足：
1、残留效应(Carry-over effects)  即第二次测量会受到第一个处理措施的影响
2、实施处理措施的顺序可能会影响结果

方法：
1、重复衡量 two conditions
2、纵向设计 longitudinal
3、验前验后 Pre-test,post-test


独立样本 Independent samples
针对受试者间设计

好处与不足就与 相依样本 相反

方法：
1、实验性检验 Experimental test
2、观察性检验 Observational test
